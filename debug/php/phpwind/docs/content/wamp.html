Wamp 简介

为了帮助 phpwind 站长实现快速建站的梦想，phpwind 开发组倾力推出一个快速架设 web 环境的软件集成包:phpwind Wamp。
您无须任何复杂操作，只要下载phpwind Wamp软件包，点击安装，就能轻松实现站长的梦想。


phpwind Wamp优点

简单:真正做到任何人都可以轻松架设论坛及运行环境.只需下载phpwind Wamp软件包，1分钟点击安装完成。
高效:phpwind Wamp 能够根据用户计算机自动配置和方便优化相关环境。

安全：
不改动您本地计算机的任何设置和文件，保证您的计算机安全正常。
采用最新的论坛程序,我们对此进行了优化和配置,论坛本身具有一定的安全性。
集成独立的卸载程序，在保留数据的前提下，不留下任何冗余文件。

运行环境需求

操作系统：Microsoft Windows2000/XP/2003

集成软件

phpwind 社区程序[简体中文] phpMyAdmin 3.1.1 ZendOptimizer 3.3.3 MySQL 5.1.30 php 5.2.8 Apache 2.2.11

安装使用

  1. 下载 phpwind Wamp 软件包
    phpwind Wamp 软件包下载地址（Windows运行环境下载）：http://www.phpwind.com/index.php?m=download
    （已更新到最新版本，最新的软件包集成最新的论坛程序）
  2. 解压缩至任意位置，执行phpwind-wamp-4.9.exe文件，根据提示进行安装。
  3. 相关帐户说明
    完成集成包安装过程后，就可以访问建设自己的PW论坛了。
    论坛默认创始人账号: admin 密码 :admin
    MySQL 数据库用户名: root 密码: phpwind.net
    （建议您安装完后修改论坛及MySQL账号密码等信息）
