论坛设置

站点默认风格
选择论坛模式默认风格。用户在没有指定风格的时候，均使用默认风格
左右分栏设置
左右分栏的开关，开启后可选择分栏的默认模式
文字广告列数
设置文字广告位的显示列数。0表示不限制，只有1列
副版主功能
  • 开启
  • 关闭
开启后，可在前台的[版块管理]添加副版主。第一版主有权任命和罢免其他版主
版块快速跳转
  • 开启
  • 关闭
开启后，可以在帖子列表页、阅读页快速跳转到别的版块
回收站功能
  • 开启
  • 关闭
开启后，删除的主题和回复将暂存于回收站
宽窄屏切换
  • 开启
  • 宽屏
  • 窄屏
勾选开启，用户可以在前台控制宽窄屏显示；勾选宽屏，则前台的显示模式为宽屏，用户不可切换；勾选窄屏，则前台的实现模式为窄屏，用户不可切换。
