什么是PWBuilder

PWBuilder是phpwind为客户提供的免费站点数据转换系统。PWBuilder是一个统一的转换平台，内部包含目前主流的转换子程序。使用它您 可以非常方便的从第三方社区软件转换数据到 phpwind 最新版本。PWBuilder倡导免费、开放的理念，我们欢迎广大站长都参与到转换程序的开发和完善中来，为PWBuilder的发展添砖加瓦。


为什么要用PWBuilder

  1. 统一平台，不用再到处寻找相应的转换程序
  2. 全部配置项目均在页面内实现。免去手工改动文件的麻烦
  3. 转换过程自动执行，无需人工干涉
  4. 自定义转换模板

PWBuilder能转换哪些论坛

PWBuilder 1.2.5 包含discuznt25_access、discuznt25_mssql、discuznt26_mssql、dvbbs82_access、dvbbs82_sql、molyx2.8.0、phpbb3.0、leobbs、bbsxp2008_access、ipb2、leadbbs50_access、Discuz6.1、Discuz7.0 、Discuz7.1 、Discuz7.2 、Discuz X1、Discuz X1.5、Discuz X2.0 到PHPWIND 8版本的转换程序。
